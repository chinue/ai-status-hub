# 变量命名规则（C/C++/CUDA）

## 指针变量

- **通用指针**：变量名以 `p` 开头（例如 `pSrc`、`pDst`、`pData`）。
- **文件指针**：变量名以 `fp` 开头（例如 `fpLog`、`fpIn`、`fpOut`）。

## 非指针变量

- **避免**使用 `p` 作为前缀，以防止与指针命名混淆（例如 `pos` 可接受，但不应使用 `pCount`/`pIndex`）。
- 变量名**应以小写字母开头**（`camelCase` 或 `snake_case` 均可接受），减少与某些全大写 / 首字母大写的宏发生潜在冲突。

## 例外

- 类型名、枚举名和宏名遵循其各自的现有约定，不受此规则约束。
